```typescript
export class ContextService {
  static WINDOW_SIZE = 20;
  static REPETITION_WINDOW = 5;
  static REPETITION_THRESHOLD = 0.85;

  static async buildContext(
    db: DrizzleDB,
    roomId: string,
    agent: { id: string; promptRole: string; promptPersonality?: string | null; promptRules?: string | null; promptConstraints?: string | null }
  ): Promise<{ systemPrompt: string; messages: Array<{ role: 'user' | 'assistant'; content: string }> }>
  // ... implementation per RESEARCH.md Pattern 2

  static async detectRepetition(db: DrizzleDB, roomId: string): Promise<boolean>
  // ... implementation per RESEARCH.md
}
```

IMPORTANT: Accept `db` as a parameter (dependency injection) instead of importing the singleton directly. This allows tests to pass an in-memory test database.

The `buildContext` method:
1. Build systemPrompt by joining non-null prompt fields with `\n\n`
2. Query `messages` table: `WHERE roomId = ? ORDER BY createdAt DESC LIMIT WINDOW_SIZE`
3. Reverse to chronological order
4. Map roles: current agent's messages -> 'assistant', all others -> 'user'
5. Return `{ systemPrompt, messages }`

The `detectRepetition` method:
1. Query last REPETITION_WINDOW messages
2. If fewer than REPETITION_WINDOW, return false
3. Compare last message against each previous via Jaccard similarity on word token sets
4. Return true if any pair exceeds REPETITION_THRESHOLD

Helper functions `tokenSet(text)` and `jaccardSimilarity(a, b)` as module-private functions.

`tokenSet`: lowercase, split on whitespace, filter tokens with length > 2, return Set.
`jaccardSimilarity`: intersection size / union size, return 0 if union is empty.
