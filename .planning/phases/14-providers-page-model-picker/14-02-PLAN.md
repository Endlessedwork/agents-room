Create `src/app/api/providers/[provider]/models/route.ts` with:

1. **ModelInfo interface** (exported for client reuse):
```typescript
export interface ModelInfo {
  id: string;
  contextLength?: number;
  capabilities?: string[];
}
```

2. **GET handler** following existing route pattern:
   - `export const dynamic = 'force-dynamic';`
   - `await params` for Next.js 16 async params
   - Validate provider against VALID_PROVIDERS array
   - Query `providerKeys` table via `db.select().from(providerKeys).where(eq(providerKeys.provider, providerName))` — use `eq` from `drizzle-orm`
   - If no API key found AND provider is not 'ollama', return `{ error: 'Not configured' }` with status 400
   - Create AbortController with 5-second setTimeout for abort
   - Call provider-specific fetch function
   - Sort results by `id` alphabetically: `models.sort((a, b) => a.id.localeCompare(b.id))`
   - Return `{ models: ModelInfo[] }`
   - On AbortError: return `{ error: 'Timeout' }` with status 504
   - On other errors: return `{ error: 'Fetch failed' }` with status 502
   - Always `clearTimeout(timeout)` in both success and error paths

3. **Five provider adapter functions** (all private, inside the same file):

**fetchAnthropicModels(apiKey: string, signal: AbortSignal): Promise<ModelInfo[]>**
- GET `https://api.anthropic.com/v1/models?limit=100`
- Headers: `{ 'X-Api-Key': apiKey, 'anthropic-version': '2023-06-01' }`
- Response shape: `{ data: [{ id, max_input_tokens?, capabilities?: { image_input?: { supported: boolean }, thinking?: { supported: boolean } } }] }`
- Map: `id` from `m.id`, `contextLength` from `m.max_input_tokens`, capabilities array: push `'vision'` if `m.capabilities?.image_input?.supported`, push `'thinking'` if `m.capabilities?.thinking?.supported`

**fetchOpenAIModels(apiKey: string, signal: AbortSignal): Promise<ModelInfo[]>**
- GET `https://api.openai.com/v1/models`
- Headers: `{ 'Authorization': \`Bearer ${apiKey}\` }`
- Response shape: `{ data: [{ id, owned_by }] }`
- Filter: `m.id.startsWith('gpt-') || /^o\d/.test(m.id)` — excludes embeddings, whisper, dall-e, tts
- Map: `id` from `m.id` only (OpenAI API returns no capability info)

**fetchGoogleModels(apiKey: string, signal: AbortSignal): Promise<ModelInfo[]>**
- GET `https://generativelanguage.googleapis.com/v1beta/models?key=${apiKey}`
- No auth headers (key in URL)
- Response shape: `{ models: [{ name, inputTokenLimit?, supportedGenerationMethods? }] }`
- Filter: only models where `supportedGenerationMethods?.includes('generateContent')`
- Map: `id` from `m.name.replace('models/', '')` (strip prefix), `contextLength` from `m.inputTokenLimit`

**fetchOpenRouterModels(apiKey: string, signal: AbortSignal): Promise<ModelInfo[]>**
- GET `https://openrouter.ai/api/v1/models`
- Headers: `{ 'Authorization': \`Bearer ${apiKey}\` }`
- Response shape: `{ data: [{ id, context_length?, architecture?: { input_modalities?: string[] } }] }`
- Map: `id` from `m.id`, `contextLength` from `m.context_length`
- Capabilities: push `'vision'` if `m.architecture?.input_modalities?.includes('image')`, push `'${Math.round(m.context_length / 1000)}k'` if `m.context_length >= 100000`

**fetchOllamaModels(baseUrl: string | null, signal: AbortSignal): Promise<ModelInfo[]>**
- Derive host: `const host = (baseUrl ?? 'http://localhost:11434').replace(/\/api$/, '')`
- GET `${host}/api/tags`
- No auth headers
- Response shape: `{ models: [{ name }] }`
- Map: `id` from `m.name` only

4. **Dispatcher function** `fetchModelsForProvider(provider: string, keyRow: {...} | undefined, signal: AbortSignal)`:
   - Switch on provider string
   - For anthropic/openai/google/openrouter: call respective adapter with `keyRow!.apiKey!`
   - For ollama: call `fetchOllamaModels(keyRow?.baseUrl ?? null, signal)`
   - Default: throw new Error(`Unknown provider: ${provider}`)
