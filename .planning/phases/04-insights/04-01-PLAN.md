1. Add new state field `tokenTotals: { input: number; output: number }` initialized to `{ input: 0, output: 0 }` in the default state.

2. Add a helper function OUTSIDE the store (exported, pure function) called `formatTokenCount(n: number): string` that:
   - Returns "0" for 0
   - Returns the raw number as string for values under 1000 (e.g., "542")
   - Returns "X.Xk" for values >= 1000 (e.g., 12400 -> "12.4k", 1000 -> "1.0k", 99500 -> "99.5k")
   - Uses `Math.round(n / 100) / 10` for the k calculation

3. Add action `updateTokenTotals: (inputTokens: number, outputTokens: number) => void` that adds the provided values to the running totals:
   ```
   set(state => ({
     tokenTotals: {
       input: state.tokenTotals.input + inputTokens,
       output: state.tokenTotals.output + outputTokens,
     }
   }))
   ```

4. Modify `loadHistory` to compute initial token totals from loaded messages by summing all non-null `inputTokens` and `outputTokens` values after mapping messages, and setting `tokenTotals` in the same `set()` call that sets messages.

5. Modify `reset` to also reset `tokenTotals` to `{ input: 0, output: 0 }`.

Do NOT modify completeTurn — the SSE hook will call updateTokenTotals separately.
